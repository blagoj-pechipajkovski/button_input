# Button

## Description

An Arduino library for reading a button with functions for: on click, on hold, and on release.

```
               [hold_wait]                [hold_interval]           [hold_interval]
    | ----------------------------- | ----------------------- | ----------------------- | -------------- | 
[on_press]                      [on_hold]                 [on_hold]                 [on_hold]      [on_release]
```
